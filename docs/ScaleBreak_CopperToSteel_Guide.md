## How to Identify Scale Break on Your Piano

### Step-by-Step Visual Inspection

1. **Open the piano**
   - Grand piano: Lift the lid
   - Upright: Remove front panel or open top lid

2. **Look at the strings from bass (left) to treble (right)**

3. **Identify the wound section** (Bass):
   ```
   Appearance: ????????  (Thick, copper-colored, rough texture)
   Feel: Heavy, stiff
   Visual: Can see copper wire spirals
   ```

4. **Find the transition point** (Scale Break):
   ```
   Last wound string:  ????  ? E2, E3, or nearby note
   First plain string: ???? ? F2, F3, or nearby note
                       ?
                   Scale Break
   ```

5. **Confirm the plain steel section** (Tenor/Treble):
   ```
   Appearance: ??????  (Thin, silver/gray, smooth)
   Feel: Light, flexible
   Visual: Uniform single wire
   ```

6. **Count piano keys to determine MIDI note**
   - Start from A0 (leftmost key on 88-key piano)
   - A0 = MIDI 21
   - Each key = +1 MIDI number
